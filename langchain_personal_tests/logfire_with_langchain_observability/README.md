## Logfire with LangChain observability

This is a simple example of how to use Logfire with LangChain observability.

### Setup

1. Create a Logfire account and get your API key.
2. Create a LangChain account and get your API key.
3. Set the following environment variables:
   ```
   LOGFIRE_API_KEY=your_logfire_api_key
   LANGCHAIN_API_KEY=your_langchain_api_key
   ```
4. Run the example:
   ```
   python main.py
   ```
