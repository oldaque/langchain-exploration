# LangChain Exploration

Este repositório é destinado à exploração, aprendizado e testes utilizando o **LangChain**. O foco principal é acompanhar as atualizações da biblioteca (versão 1.0+) e experimentar suas novas funcionalidades.

## Estrutura do Repositório

Atualmente, o repositório contém materiais de estudo prático:

- **langchain_academy/**: Diretório dedicado a cursos e tutoriais.
  - **lca-langchainV1-essentials/**: Curso de fundamentos da versão 1.0, cobrindo tópicos essenciais como:
    - Agentes Rápidos (Fast Agents)
    - Mensagens e Streaming
    - Ferramentas (Tools) e integração com MCP (Model Context Protocol)
    - Memória e Persistência
    - Saídas Estruturadas (Structured Output)
    - Configurações Dinâmicas
    - Human-in-the-loop (HITL)

## Objetivos

- Aprender e dominar a API atualizada do LangChain.
- Criar exemplos práticos e provas de conceito (PoCs).
- Explorar integrações com diferentes modelos e ferramentas.

## Como usar

Navegue pelos diretórios de cursos para encontrar notebooks (`.ipynb`) e scripts Python com exemplos práticos. Certifique-se de instalar as dependências listadas nos arquivos `requirements.txt` ou `pyproject.toml` de cada projeto.
